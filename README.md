# chinook_analytics_sqLite3

Bu proje, örnek Chinook veritabanı üzerinde birleşik (join'li) sorguların çalıştırılmasını içerir.
Kullanılacak tablolar: tracks, albums, artists, genres.

Yapı

chinook.db: SQLite veritabanı

sql/queries.sql: SQL sorguları

src/run_queries.py: Sorguları çalıştırmak için Python betiği (script)

README.md: Kılavuz

Çalıştırma

Bash

cd chinook_analytics_sqLite3
python3 src/run_queries.py
